# odin-etch-a-sketch

This project is something between a sketch pad and an Etch A Sketch. It has some features like progressive darkening effect, change grid size, and random colors. It was developed as part of [The Odin Project (TOP)](https://www.theodinproject.com/) curriculum, with the goal of practicing DOM manipulation skills.

The project is deployed via GitHub Pages, so you can easily [try it out here](https://pedrody.github.io/odin-etch-a-sketch/).
